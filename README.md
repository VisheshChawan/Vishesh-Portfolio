# Vishesh Chawan — Portfolio

My personal developer portfolio built with Next.js, React, TypeScript, and Framer Motion.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **UI:** React 19, Framer Motion, vanilla CSS
- **State:** Zustand (persisted to localStorage / IndexedDB)
- **Fonts:** Space Grotesk, Fira Code, Teko
- **Deployment:** Vercel

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
